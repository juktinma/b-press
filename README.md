# B-Press

B-Press 是一个轻量级、优雅、注重隐私与安全的博客系统。基于 Node.js 开发，支持 **SQLite** 与 **MySQL** 双数据库引擎。系统提供了完整的文章、独立页面、分类、评论、标签管理功能，并且支持多主题切换与深度的前端自定义设置。

## ✨ 特性 (Features)

- **双数据库支持**：完美支持轻量级的 SQLite（免安装直接使用）与高性能的 MySQL，自由选择。
- **极致隐私与安全**：
  - **本地化依赖**：完全剥离外部 CDN 依赖（如前端资源、编辑器库），支持离线内网环境，消除第三方请求隐私泄漏。
  - **安全加固**：防范 XSS、CSRF、SQL 注入与路径遍历攻击，采用更健壮的 JWT 鉴权与密码哈希方案。
- **现代化后台与写作体验**：集成 EasyMDE 提供沉浸式的 Markdown 写作体验，支持拖拽上传图片、实时预览。
- **精美多主题支持**：内置 `Default`（极简硬朗风）和 `Sea Foam`（海洋毛玻璃风）两款精美主题，支持一键切换与独立配置。
- **丰富的定制选项**：支持多种头图排版模式、文章轮播组件配置，以及丰富的挂件与侧边栏。

## 🛠️ 安装与运行 (Installation & Usage)

### 环境要求
- Node.js >= 18.0

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/juktinma/b-press.git
   cd b-press
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动服务器**
   ```bash
   npm start
   ```
   *说明：系统将默认运行在 `3000` 端口。推荐使用 `pm2` 或其他进程管理工具进行生产环境部署。*

4. **初始化安装**
   打开浏览器访问 `http://localhost:3000`，系统将自动引导进入安装界面，设置管理员账号密码及博客基本信息。您可以在安装过程中选择使用 SQLite 或 MySQL 数据库。

## 📂 目录结构 (Directory Structure)

- `/admin`：管理后台的前端页面及静态资源。
- `/data`：SQLite 数据库及配置文件的存储目录（自动生成）。
- `/public`：系统公共静态资源（上传的图片、附件、本地化的前端依赖包等）。
- `/routes`：后端 API、路由中间件与安全过滤器。
- `/themes`：前台主题目录，每款主题具有独立的配置、视图（EJS）及 CSS 静态资源。
- `server.js`：应用程序的入口文件。
